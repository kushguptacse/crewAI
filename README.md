## CrewAI
### CrewAI Flows: 
Flows allow you to create structured, event-driven workflows that manage state and control execution.
### CrewAI Crews: 
Crews are teams of autonomous agents that collaborate to solve specific tasks delegated to them by the Flow.

## Core concepts:
1. Agent : An autonomous unit, with an LLM, a role, a goal, a back-story, memory, tools.
2. Task : A specific assignment to be carried out with a description, expected output, agent.
3. Crew: A team of agents and tasks, which can run in sequential or hierarchical using a manager LLM to assign.
